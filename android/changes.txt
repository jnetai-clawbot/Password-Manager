# Android Password Manager - Changes Summary

## Version 1.1.0 (Updated to match Python/C versions)

### Features Added (matching Python/C versions):
1. **Master Password Authentication** - Setup and login with SHA-256 hashed password
2. **Add Entry** - Site, username, password, URL, notes, category
3. **List All Entries** - Scrollable list with search and category filter
4. **Get Password** - Search by site name to retrieve credentials
5. **Edit Entry** - Modify all fields including category
6. **Delete Entry** - Remove entries by site name
7. **Generate Password** - Configurable length (8-64 chars)
8. **Export JSON** - Backup all entries to JSON file
9. **Import JSON** - Restore entries from JSON backup
10. **Lock/Logout** - Lock app requiring password re-entry

### UI Features:
- Search bar filters entries by site/username/URL in real-time
- Category spinner to filter by category (general, work, personal, finance, social, other)
- Results count showing filtered/total entries
- Entry cards show category chip
- Copy Username and Copy Password buttons
- Material Design 3 dark theme

### Database:
- Room database with category field
- Version 2 (may require migration for existing users)
- Supports search queries across site, username, URL

### Menu Options:
- Get Password - Quick lookup by site
- Generate - Password generator dialog
- Delete - Delete by site name
- Import/Export - JSON backup/restore
- Lock - Lock app
- About - App info

### GitHub Actions:
- `.github/workflows/android.yml` builds debug APK on push
- APK output to `apk/` directory
