### Build Instructions

**To compile the GUI:**
```bash
cd /home/jay/Documents/Scripts/AI/openclaw/job17/C
make gui
```

**Exact gcc command used:**
```bash
gcc -Wall -Wextra -O2 -std=c11 -o gui/password_manager_gui gui/password_manager_gui.c \
    -lcrypto -lsqlite3 \
    -I/usr/include/gtk-3.0 -I/usr/include/pango-1.0 -I/usr/include/glib-2.0 \
    -I/usr/include/harfbuzz -I/usr/include/cairo \
    -I/usr/include/gdk-pixbuf-2.0 -I/usr/include/atk-1.0 \
    -I/usr/lib/aarch64-linux-gnu/glib-2.0/include \
    -lgtk-3 -lgdk-3 -lpangocairo-1.0 -lpango-1.0 -lharfbuzz \
    -latk-1.0 -lcairo-gobject -lcairo -lgdk_pixbuf-2.0 -lgio-2.0 \
    -lgobject-2.0 -lglib-2.0
```

**To run:**
```bash
cd /home/jay/Documents/Scripts/AI/openclaw/job17/C
rm -f passwords.db  # Optional: start fresh
./gui/password_manager_gui
```
Running in Debug Mode:

./password_manager_gui 2>&1 | head -50

Or
./password_manager_gui 2>&1 | grep -i "debug"
