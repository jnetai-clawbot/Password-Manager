## Issue: GUI not showing or not responding

### Cause
The database already has a master password set. When you click "Reset" and set a new password, it fails because the old password is still there.

### Solution
Use CLI to reset:
```bash
rm ~/.passwords.db
python3 password_manager_gui.py
```

Or use the menu version which works without display:
```bash
python3 password_manager_menu.py
```
